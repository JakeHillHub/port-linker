# Port Linker
Use this to link any combination of tcp sockets and serial ports together

# Requirements
pyserial
funcy
pydux

# Usage (Install)
1. `pip install port-linker`
2. `port-linker <configuration.json> [--wait <seconds>]`

# Usage (Git clone)
`python -m core.run_link config.json [--wait <seconds>]`
